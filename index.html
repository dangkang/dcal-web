なぜ「ローカル完結」か

一般的なカレンダーアプリdCal
予定がクラウドサーバーへ送信されるデバイス内のみ。ネット通信なし
AI 学習・広告ターゲティング対象になる可能性学習データに使われません
アカウント乗っ取り = 全予定流出のリスク物理的にあなたの Mac/iPhone 内のみ
サブスク終了でデータ閉じ込めプレーンテキストで永久保存
月単位表示。月またぎの予定は分断4 週連続グリッド
予定はタイトル中心1 日に長文メモ＋装飾

「ローカル完結」は物理的に証明されています

$ codesign -d --entitlements - dCal.app

  com.apple.security.app-sandbox          ← 有効
  com.apple.security.files.user-selected  ← 有効（ファイル選択時のみ）

  com.apple.security.network.client       ← 不存在
  com.apple.security.network.server       ← 不存在

macOS の App Sandbox がネットワーク権限を許可していないため、 dCal は外部サーバーへ接続したくてもできません。

特長

4 週グリッドで月またぎが見える

3〜6 週まで切り替え可能。期間予定はカラーラインで月境を貫いて連続表示。 Apple Calendar の月表示では分断される予定も一望できます。

1 日に長文メモが書ける

会議メモ・業務日誌・家族の連絡事項 — 太字・斜体・下線・色付けなど Markdown 風の装飾も可能。本文全文検索で過去のメモを瞬時に発掘。

hyCalendar ファイル互換

Windows 版 hyCalendar の .cal ファイルをそのまま開けます。 バイト単位で完全互換 — Mac で編集して Windows に戻しても識別不能。

日本の祝日対応

振替休日・国民の休日も自動表示。1980〜2099 年に対応した 国民の祝日法ベースの計算ロジックを内蔵。

過去年への素早いジャンプ (⌘J)

年スクロール + 12 か月グリッドのポップオーバー。 2003 年でも 2099 年でも 2 クリックで遡れます。

全文検索 (⌘F)

日付メモ・期間予定タイトルを全件横断。マッチハイライト付き。 数千エントリのファイルでも瞬時。

Mac + iPhone — 同期はあなたの自由

iCloud Drive / Dropbox / Google Drive / OneDrive など、お好みの クラウドファイル同期サービスを利用。dCal 自体は API を叩きません。

プレーンテキストで永久保存

独自バイナリ形式ではなく、テキストとして読める形式で保存。 20 年後に dCal がなくても、データは生き続けます。

よくある質問

Q. データはどこに保存されるの?

あなたが選んだ場所だけです。デフォルトは Mac/iPhone のローカルストレージ。 iCloud Drive やサードパーティのクラウドサービスにファイルを置くこともできますが、 dCal 自体はそれらと一切通信しません。

Q. クラウド同期はないの?

はい、ありません。代わりに iCloud Drive / Dropbox / Google Drive / OneDrive 等の汎用ファイル同期サービスを使って、自分で .cal ファイルを同期できます。

Q. Windows 版 hyCalendar のファイルは開ける?

はい、バイト単位で完全互換です。実ファイル (3MB+ / 7,000日項目 / 1,700期間予定) でも問題なく開けます。

Q. AI 学習データに使われる?

使われません。dCal はネットワーク通信を持たないため、どんなクラウドサービスや AI モデルにも一文字も送られません。

Q. 法人向けライセンスはある?

2027 年に対応予定です。お問い合わせください。
